ANOVA

¬øQu√© es el An√°lisis de Varianza (ANOVA)?

Es una prueba estad√≠stica que permite comparar la media (promedio) de m√°s de 2 grupos. Es decir, es como una prueba T, pero de m√°s de dos grupos.

Al igual que con la prueba T, en esta oportunidad debemos observar el p-value de la prueba. Para ello, se establecen 2 hip√≥tesis:

H0 (p-value > 0.05): no hay diferencia significativa entre las medias de ning√∫n grupo. H1 (p-value =< 0.05): al menos una media poblacional (de un grupo) es significativamente distinta a las otras

Estad√≠stico F

Indica la variablidad entre grupos y dentro de un mismo grupo.

Si F es grande, hay mucha variabilidad entre grupos y poca variabilidad dentro de un mismo grupo. Si F es peque√±o, hay poca variabilidad entre grupos pero mucha variabilidad dentro de un mismo grupo.

Prueba Tukey

Prueba ‚Äúpost-hoc‚Äù (luego de), realizada para revisar entre qu√© grupos existe una diferencia significativa.

OJO: como es una prueba complementaria a la ANOVA, solo resulta necesaria si se rechaza la H0 en ANOVA. Si la H0 no se rechaza en ANOVA, no hay necesidad de hacer prueba Tukey.

Al igual que las otras pruebas estad√≠sticas, se debe ver el p-value en cada emparejamiento

Diagramas de dispersi√≥n y correlaci√≥n

Ideas clave

La correlaci√≥n es en esencia una medida normalizada de asociaci√≥n o covariaci√≥n lineal entre dos variables.

  • La correlaci√≥n es una medida de la relaci√≥n (covariaci√≥n) entre dos variables cuantitativas.

  • La manera m√°s sencilla de saber si dos variables est√°n correlacionadas es determinar si co-var√≠an (var√≠an conjuntamente).

  • Es importante hacer notar que esta covariaci√≥n o relaci√≥n no implica necesariamente causalidad: La correlaci√≥n puede ser fortuita, como en el caso cl√°sico de la correlaci√≥n entre el n√∫mero de venta de helados e incendios, debido al efecto de una tercera variable, la temperatura ambiental. A este tipo de relaci√≥n se le llama ‚Äúespuria‚Äù.

Hip√≥tesis de la prueba de correlaci√≥n

  • H0 : No existe correlaci√≥n entre las variables
  • H1 : Existe correlaci√≥n entre las variables

Coeficiente de Correlaci√≥n de Pearson

  • ‚ÄúEl Coeficiente de Correlaci√≥n de Pearson es un estad√≠stico param√©trico, pues se asume que ambas variables tienen una distribuci√≥n aproximadamente normal, o sea, distribuci√≥n normal bivariante‚Äù.

  • Es una medida que puede variar entre -1 y +1, ambos extremos indicando correlaciones perfectas, negativa y positiva respectivamente.

  • Un valor de r = 0 indica que no existe relaci√≥n lineal entre las dos variables.
